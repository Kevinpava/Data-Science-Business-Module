# Módulo 1 — Fundamentos de Data Science y Programación con Python

## 🎯 Objetivo del Módulo

Sentar las bases de lo que significa hacer Data Science en una empresa, dominando Python como herramienta principal para el análisis de datos y la automatización de tareas.

---

## 📚 Temario

### 1. Introducción a Data Science en la empresa
- Roles en un equipo de datos (Data Engineer, Data Scientist, Data Analyst, MLOps)
- KPIs de negocio vs KPIs técnicos
- Ciclo de vida de un proyecto (CRISP-DM adaptado)

### 2. Python para análisis de negocio
- Variables y tipos de datos
- Listas, tuplas y diccionarios aplicados a información comercial
- Manejo de datos básicos

### 3. Control de flujo aplicado a reglas de negocio
- Condicionales if/else para validar transacciones
- Bucles para procesar grandes volúmenes de datos

### 4. Funciones para automatizar reportes
- Creación de funciones reutilizables
- Automatización de informes periódicos

### 5. NumPy: cálculos numéricos para finanzas y operaciones
- Arrays y operaciones vectorizadas
- Estadística básica con NumPy

### 6. Pandas básico: Series y DataFrames
- Manejo de datos reales (ventas, clientes, inventarios)
- Exploración inicial de datos

### 7. Limpieza inicial de datos empresariales
- Tratamiento de valores nulos
- Eliminación de duplicados
- Corrección de formatos inconsistentes

### 8. Lógica de negocio y definición de problemas analíticos
- Cómo traducir preguntas de negocio a problemas de datos
- Definición de alcance y éxito

### 9. Estructuración de un proyecto DS en la empresa
- Organización de carpetas
- Control de versiones con Git
- Documentación profesional

### 10. Caso práctico interno
- Análisis completo de un dataset de ventas o transacciones
- Aplicación de todo lo aprendido

---

## 🛠️ Tecnologías utilizadas

- Python 3.x
- NumPy
- Pandas
- Jupyter Notebooks
- Git / GitHub
